#### 1. Docker 이미지 빌드
```
docker build -t my-fastapi-app .
```

#### 2. 컨테이너 실행
```
docker run -d -p 8000:8000 --name fastapi-app my-fastapi-app
```

#### 테스트 방법
* http://localhost:8000/
* http://localhost:8000/hello/World
* http://localhost:8000/health
* http://localhost:8000/info

#### 컨테이너 내부로 들어가기
```
docker exec -it fastapi-app sh
```
